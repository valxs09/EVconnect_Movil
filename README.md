# EVCONNECT

EVCONNECT es una aplicación Flutter diseñada para facilitar la búsqueda y reserva de estaciones de carga para vehículos eléctricos. La aplicación permite a los usuarios registrarse, iniciar sesión y gestionar sus reservas de carga de manera eficiente.

## Estructura del Proyecto

- **lib/main.dart**: Punto de entrada de la aplicación.
- **lib/config/routes.dart**: Define las rutas de navegación.
- **lib/config/theme.dart**: Configuración del tema de la aplicación.
- **lib/models/**: Contiene los modelos de datos para usuarios, estaciones y reservas.
- **lib/screens/**: Pantallas de la aplicación, incluyendo autenticación, inicio, estaciones y reservas.
- **lib/services/**: Lógica de negocio para autenticación, estaciones y reservas.
- **lib/widgets/**: Componentes reutilizables de la interfaz de usuario.
- **lib/utils/constants.dart**: Constantes utilizadas en toda la aplicación.
- **pubspec.yaml**: Configuración del proyecto Flutter.

## Instalación

Para ejecutar la aplicación, clona este repositorio y ejecuta:

```bash
flutter pub get
flutter run
```
